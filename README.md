## Zenith: An Opinionated Laravel Starter Kit

This starter kit installs Laravel with the following packages:

- [Laravel Livewire](https://livewire.laravel.com/)
- Tailwind CSS

**Dev Packages**
- Spatie's Ray
- Larastan
- Pint

**Additional packages**
- Laravel Pulse

### Installation / Configuration

- `bash install.sh`
